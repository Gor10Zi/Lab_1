# Lab_1
1. Короткий опис задачі
Програма повинна:
Ввести з клавіатури 5 елементів масиву.
Знайти найбільший елемент масиву.
Відсортувати масив за спеціальним правилом:
елементи з парними індексами впорядковуються за зростанням;
елементи з непарними індексами впорядковуються за спаданням.
Вивести:
найбільший елемент масиву;
відсортований масив. (Для сортування використовується метод бульбашки)

3. Опис алгоритму розв’язання
Алгоритм роботи програми складається з кількох послідовних етапів.
Крок 1. Оголошення змінних
Створюється:
константа n = 5 — розмір масиву;
масив a[n] для зберігання чисел;
допоміжні змінні:
i, j — лічильники циклів;
temp — для обміну елементів;
max — для збереження найбільшого елемента.
Крок 2. Введення масиву
За допомогою циклу for користувач вводить 5 елементів масиву.
Алгоритм:
цикл виконується від i = 0 до i < n
кожен елемент вводиться з клавіатури
значення зберігається у a[i].
Крок 3. Пошук найбільшого елемента
найбільшим тимчасово вважається перший елемент масиву: max = a[0]
програма порівнює його з іншими елементами масиву.
якщо знаходиться більше значення — змінна max оновлюється.
Крок 4. Сортування масиву
Для сортування використовується метод бульбашки.
Алгоритм:виконується декілька проходів по масиву;
порівнюються сусідні елементи;
якщо вони стоять у неправильному порядку — вони міняються місцями.
Особливість цієї задачі:якщо індекс j парний, то елементи впорядковуються за зростанням,якщо індекс j непарний, то елементи впорядковуються за спаданням,
Для обміну використовується змінна temp.
